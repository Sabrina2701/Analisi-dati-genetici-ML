# Analisi-dati-genetici-ML

Questo progetto di tesi si focalizza sull'integrazione tra informatica avanzata e biomedicina nel contesto della medicina di precisione. L'obiettivo principale è lo sviluppo di un algoritmo di Machine Learning in grado di predire lo stato vitale di un paziente (vivo o deceduto) partendo dalle misurazioni dell'espressione genica di tipo TPM (Transcripts Per Million).
